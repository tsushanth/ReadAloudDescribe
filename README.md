# ReadAloud Describe

On-device AI image description for Android.

The 90 % of Android users without a Pixel 8+ get nothing useful when TalkBack hits an unlabeled image. This app fills that gap — runs entirely on the device, works offline, never sends pixels anywhere. Reads descriptions back in whichever voice the user picked in **ReadAloud Voice**.

- **Status**: planning. See [PROJECT_PLAN.md](./PROJECT_PLAN.md).
- **Model**: Moondream2 INT8 ONNX
- **Distribution**: Google Play
- **Application ID**: `com.listenai.describe`
- **Sibling products**: [ReadAloud AI](../ReadAloudAI/) (reader) · ReadAloud Voice (TTS engine)
